#### env_is64BitProcess()

現在のプロセスが64ビットプロセスであるかどうかを示す値を取得します。

```cs title="AliceScript"
namespace Alice.Environment;
public bool env_is64BitProcess();
```

|戻り値| |
|-|-|
|`bool`|プロセスが64ビットの場合は`true`、それ以外の場合は`false`。|
