#### math_isFinite(number)

指定した数値が有限であるかを表す値を取得します。

```cs title="AliceScript"
namespace Alice.Math;
public bool math_isFinite(number value);
```

|引数| |
|-|-|
|`value`|判定対象の数値。|

|戻り値| |
|-|-|
|`number`|`value`が有限の値であれば`true`、それ以外の場合は`false`。|
