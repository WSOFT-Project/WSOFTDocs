#### Size()

現在の配列にある要素の数を取得します

```cs title="AliceScript"
class Alice.Array;
public number Size();
```

|戻り値| |
|-|-|
|`number`| 現在の配列にある要素の数|

???note "対応: AliceScript RC2以降"
    |対応||
    |---|---|
    |AliceScript|RC2、GM、2.0、2.1、2.2、2.3、3.0、4|
    |AliceSister|GM、2.0、2.1、2.2、2.3、3.0、4|
    |Losetta|0.8、0.9、0.10、0.11|

### 例
以下は、`abc`の文字数を表示します

```cs title="AliceScript"
var a = "abc";
print(a.Size()); // 出力例 : 3
```
